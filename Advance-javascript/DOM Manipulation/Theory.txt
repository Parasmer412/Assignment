(Q_1) :What is the DOM (Document Object Model) in JavaScript? How does JavaScript interact with the DOM?

=> DOM

> The DOM is a programming interface for web documents.

> It represents the structure of a webpage as a tree of objects.

> Each element (like <p>, <div>, <h1>, etc.) is a node in the tree.

> With the DOM, JavaScript can access, modify, add, or delete elements and their content   dynamically.

> Think of the DOM as the bridge between HTML and JavaScript.



=> How JavaScript Interacts with the DOM


> Access elements

     document.getElementById("id")
    
     document.querySelector("selector")

> Change content

       element.innerHTML = "New text";
      
      element.textContent = "New text";

> Change styles

      element.style.color = "red";
      
      Add or remove elements
      
      document.createElement("div")
      
      parent.appendChild(child)
      
      element.remove()

Handle events

       element.addEventListener("click", function() { ... });

(Q-2): Explain the methods getElementById(), getElementsByClassName(),and querySelector() used to select elements from the DOM.

  1. getElementById()

> Selects a single element by its unique id.

> Returns the element if found, otherwise null.

> Since IDs must be unique, it always returns one element.

> Syntax: document.getElementById("idName");


> 2. getElementsByClassName()

> Selects all elements that share the same class name.

> Returns an HTMLCollection (like an array but not exactly).

> You can loop through it to access each element.

> Syntax: document.getElementsByClassName("className");

 
 3. querySelector()

> Selects the first element that matches a CSS selector (like #id, .class, tag).

> More flexible than the others because it accepts any CSS selector.

> Returns a single element or null if not found.

 Syntax: document.querySelector("CSS_selector");
